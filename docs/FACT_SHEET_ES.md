# 📑 FICHA TÉCNICA: PASSLY v3.0 (Cloud Edition)
**Sistema de Control de Accesos Inteligente**

---

## 🔝 RESUMEN EJECUTIVO
Passly es una solución integral para la gestión de identidades y seguridad física en entornos residenciales y corporativos. Transforma el registro manual de entradas y salidas en un proceso digitalizado, auditable y de alta velocidad mediante el uso de tecnologías web modernas y códigos QR dinámicos.

## 🎯 OBJETIVOS CLAVE
*   **Digitalización Total**: Eliminar el uso de bitácoras físicas en porterías.
*   **Seguridad Multi-Capa**: Implementar estándares de autenticación bancaria (MFA/TOTP).
*   **Monitoreo Real**: Proporcionar visibilidad instantánea del flujo de personas a los administradores.
*   **Autogestión**: Permitir que los residentes gestionen sus propias invitaciones y dispositivos.

## 🛠️ STACK TECNOLÓGICO
*   **Backend**: Node.js + Express (API RESTful).
*   **Frontend**: Vanilla JavaScript (SPA) + CSS3 Glassmorphism.
*   **Base de Datos**: MySQL 8.0 Cloud (Aiven) con integridad referencial 3FN.
*   **Comunicación**: WebSockets (Socket.IO) para actualizaciones en vivo.
*   **Infraestructura**: Despliegue híbrido en Render (API) y Vercel (Web).

## 🌟 FUNCIONALIDADES DESTACADAS
1.  **Magic Login (Zero-Auth)**: Acceso rápido para demostraciones y contingencias.
2.  **Sistema QR Dinámico**: QRs permanentes para residentes y temporales (JWT) para invitados.
3.  **Gestión de Equipos**: Módulo dedicado para activos tecnológicos (Laptops/Tablets).
4.  **Auditoría Forense**: Registro inmutable de IPs y acciones administrativas.
5.  **Reportes Profesionales**: Generación de informes en PDF y CSV integrados.

## 📈 VALOR AGREGADO
*   **Mobile-First**: Experiencia optimizada para tablets de guardias y móviles de residentes.
*   **Modo Nuclear**: Resiliencia total con datos mock en caso de desconexión de base de datos.
*   **Escalabilidad**: Listo para gestionar múltiples sedes desde un solo panel.
