# 📑 FACT SHEET: PASSLY v3.0 (Cloud Edition)
**Intelligent Access Control System**

---

## 🔝 EXECUTIVE SUMMARY
Passly is a comprehensive identity management and physical security solution designed for residential and corporate environments. It transforms manual entry/exit logs into a digital, auditable, and high-speed process using modern web technologies and dynamic QR codes.

## 🎯 KEY OBJECTIVES
*   **Total Digitalization**: Eliminate the use of physical logbooks at security gates.
*   **Multi-Layer Security**: Implement bank-grade authentication standards (MFA/TOTP).
*   **Real-Time Monitoring**: Provide administrators with instant visibility of people flow.
*   **Self-Management**: Allow residents to manage their own invitations and devices.

## 🛠️ TECHNOLOGY STACK
*   **Backend**: Node.js + Express (RESTful API).
*   **Frontend**: Vanilla JavaScript (SPA) + CSS3 Glassmorphism.
*   **Database**: MySQL 8.0 Cloud (Aiven) with 3FN relational integrity.
*   **Communication**: WebSockets (Socket.IO) for live updates.
*   **Infrastructure**: Hybrid deployment on Render (API) and Vercel (Web).

## 🌟 FEATURE HIGHLIGHTS
1.  **Magic Login (Zero-Auth)**: Fast access for demonstrations and contingencies.
2.  **Dynamic QR Ecosystem**: Permanent QRs for residents and temporary JWTs for guests.
3.  **Asset Management**: Dedicated module for technological assets (Laptops/Tablets).
4.  **Forensic Auditing**: Immutable logs of IPs and administrative actions.
5.  **Professional Reporting**: Integrated PDF (branded) and CSV reporting.

## 📈 ADDED VALUE
*   **Mobile-First**: Optimized experience for guard tablets and resident smartphones.
*   **Nuclear Mode**: Total resilience with mock data in case of database disconnection.
*   **Scalability**: Ready to manage multiple sites from a single dashboard.
